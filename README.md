# 🚀 C++ Project Template

## 📝 Overview

A modern C++ project template with:
- 🔨 CMake build system
- 📋 Makefile commands
- 🧪 Unit testing
- 🔍 Static code analysis
- 💡 Memory checking
- 📊 Code coverage

## 🛠 Prerequisites

```bash
sudo apt-get install -y \
    cmake \
    g++ \
    clang-format \
    clang-tidy \
    valgrind \
    lcov \
    libgtest-dev
```

## 🗂 Project Structure

```
project_root/
├── 📄 CMakeLists.txt
├── 📝 Makefile
├── 📂 src/
├── 📂 include/
└── 📂 tests/
```

## 🚦 Commands

| Command | Action |
|---------|--------|
| `make build` | 🏗 Compile project |
| `make test` | 🧪 Run tests |
| `make valgrind` | 🕵️ Check memory leaks |
| `make coverage` | 📊 Generate coverage report |
| `make format` | 🖌 Format code |
| `make tidy` | 🔍 Static analysis |

## 🚀 Quick Start

1. Clone repository
2. Install prerequisites
3. Run `make build`
4. Run `make test`
